# V2 快速启动指南

## 🚀 5 分钟快速开始

### 1️⃣ 启动服务器

```bash
npm run dev
```

等待服务器启动，看到：
```
✓ Ready in 2.5s
○ Local: http://localhost:3000
```

### 2️⃣ 打开浏览器

访问 `http://localhost:3000`

### 3️⃣ 确认 V2 版本

页面顶部应该显示：
- 标题：**AI 数据看板**
- 右上角：**v2.0 ← 切换到 v1**

如果显示 v1.0，点击切换按钮切换到 V2。

### 4️⃣ 输入需求

在输入框中输入（或点击示例按钮）：

```
一个电商销售数据仪表盘，包含 GMV 趋势图、品类占比饼图、TOP10 商品表格、关键指标卡片
```

### 5️⃣ 开始生成

点击 **"开始设计"** 按钮（或按 Cmd/Ctrl + Enter）

### 6️⃣ 等待完成

观察进度：
1. **Step 1: VI 设计** - 约 10-20 秒
2. **Step 2: 故事设计** - 与 Step 1 并发
3. **Step 3: 生成代码** - 约 30-60 秒

总时间：约 1-2 分钟

### 7️⃣ 查看结果

完成后，你会看到：

**左侧**：
- ✅ VI 系统预览（颜色、字体等）
- ✅ 看板布局预览（页面结构）
- ✅ 生成的代码

**右侧**：
- ✅ **实时渲染的看板**（这是关键！）

## ✅ 成功标志

如果看到以下内容，说明一切正常：

1. ✅ 右侧显示了实际的看板内容（不是 "等待 JSX 代码输入…"）
2. ✅ 看板包含标题、数字、图表等元素
3. ✅ 底部有页码指示器（如果是多页）
4. ✅ 可以点击页码切换页面
5. ✅ 可以点击"复制"按钮复制代码

## ❌ 如果出现问题

### 问题：右侧显示白屏

**解决方案**：
1. 打开浏览器控制台（F12）
2. 查看 Console 标签的错误信息
3. 查看 `V2_TESTING_GUIDE.md` 的"常见问题排查"部分

### 问题：API 请求失败

**解决方案**：
1. 检查 `.env.local` 文件是否存在
2. 确认 API 密钥配置正确
3. 查看 Network 标签的请求详情

### 问题：生成速度很慢

**这是正常的**：
- AI 生成需要时间
- 完整流程约 1-2 分钟
- 请耐心等待
