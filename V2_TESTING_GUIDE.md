## 常见问题排查

### 问题 1：右侧仍然显示白屏

**可能原因**：
- AI 生成的代码格式不符合预期
- 代码中有语法错误

**排查步骤**：
1. 打开浏览器控制台，查看是否有错误信息
2. 检查左侧的代码预览，确认代码是否完整
3. 查看 `[SimpleJsxRenderer]` 的日志输出

### 问题 2：API 请求失败

**可能原因**：
- `.env.local` 文件中的 API 密钥未配置
- DeepSeek API 服务不可用

**排查步骤**：
1. 检查 `.env.local` 文件是否存在
2. 确认环境变量配置正确
3. 查看 Network 标签，检查 API 请求的响应

### 问题 3：生成速度很慢

**正常现象**：
- Step 1 + Step 2 并发执行，通常需要 10-30 秒
- Step 3 生成代码，通常需要 30-60 秒
- 总时间约 1-2 分钟

### 问题 4：生成的看板样式不理想

**这是正常的**：
- AI 生成的代码质量取决于 prompt 和模型能力
- 可以点击"重试"按钮重新生成
- 可以修改输入需求，提供更详细的描述

## 技术架构总结

### V2 系统组成

```
用户输入
    ↓
┌─────────────────────────────────────┐
│  src/app/page.tsx                   │  版本切换
│  (V1 ↔ V2)                          │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│  src/components/v2/                 │
│  board-studio-v2.tsx                │  主界面
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│  src/hooks/use-pipeline-v2.ts       │  状态管理
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│  API Routes (并发)                  │
│  ├─ /api/board/v2/design-vi         │  Step 1
│  └─ /api/board/v2/design-story      │  Step 2
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│  /api/board/v2/generate-jsx         │  Step 3
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│  src/components/v2/                 │
│  simple-jsx-renderer.tsx            │  代码渲染
└─────────────────────────────────────┘
    ↓
最终看板显示
```

### 数据流

```
用户需求 (brief)
    ↓
Step 1: VISystem (颜色、字体、间距等)
Step 2: BoardStory (页面结构、内容规划)
    ↓
Step 3: JSXCode (完整的 React 组件代码)
    ↓
SimpleJsxRenderer (动态执行代码)
    ↓
渲染的看板组件
```
