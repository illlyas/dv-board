## 🔧 技术架构

### 依赖注入系统

SimpleJsxRenderer 现在注入以下依赖：

```typescript
{
  React,              // React 核心
  useState,           // 状态管理
  useEffect,          // 副作用
  useMemo,            // 记忆化
  useCallback,        // 回调记忆化
  useRef,             // 引用
  echarts,            // ECharts 库
  EChartsWrapper      // 图表包装组件
}
```

### EChartsWrapper 组件

功能：
- ✅ 自动初始化 ECharts 实例
- ✅ 响应 option 变化
- ✅ 窗口大小自适应
- ✅ 自动资源清理

使用方式：
```javascript
React.createElement(EChartsWrapper, {
  option: { /* ECharts 配置 */ },
  style: { width: 400, height: 300 }
})
```

## ⚠️ 已知限制

1. **AI 生成质量**
   - 代码质量取决于 AI 模型能力
   - 可能需要多次重试才能得到理想结果

2. **图表类型**
   - 目前支持基本图表类型（柱状图、折线图、饼图等）
   - 复杂图表（地图、3D 图表）需要额外配置

3. **交互功能**
   - 生成的图表暂不支持复杂交互
   - 工具栏、数据联动等需要手动添加

4. **性能**
   - 大数据量时可能有性能问题
   - 建议数据量控制在合理范围
