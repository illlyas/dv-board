# 🚀 快速启动指南

## V2 版本已集成！

V2 看板生成系统现在是**默认版本**，已经完全集成到应用中。

## 📦 启动应用

```bash
# 安装依赖（如果还没安装）
npm install

# 启动开发服务器
npm run dev

# 打开浏览器访问
# http://localhost:3000
```

## 🎯 立即体验 V2

### 1. 打开应用
访问 http://localhost:3000，你会看到 V2 的界面。

### 2. 输入需求
在输入框中输入你的看板需求，例如：
```
创建一个电商销售数据看板，包含销售额趋势、品类占比、TOP10商品排行
```

### 3. 等待生成
AI 会自动执行三个步骤：
- ① 设计 VI 系统（并发）
- ② 设计看板故事（并发）
- ③ 生成 JSX 代码

### 4. 查看结果
- **左侧**：查看 VI 系统、看板故事、生成的代码
- **右侧**：实时预览渲染效果

### 5. 使用代码
点击"复制代码"按钮，将生成的 JSX 代码用到你的项目中。

## 🔄 切换到 V1

如果你想使用旧版本：
1. 点击右上角的"切换到 V1 版本"按钮
2. 或者修改 `src/app/page.tsx` 中的默认版本

## 📝 示例需求

### 简单示例
```
创建一个销售数据看板
```

### 详细示例
```
创建一个电商运营监控大屏，需要展示：
1. 实时销售额和订单量（大数字 KPI）
2. 近30天销售趋势（折线图）
3. 品类销售占比（饼图）
4. TOP10 热销商品（排行榜）
5. 区域销售分布（柱状图）

使用深色科技风格，蓝色系为主色调。
目标受众是运营团队。
```

## 🎨 V2 的特点

✅ **并发执行** - Step 1 和 Step 2 同时进行，更快  
✅ **完整 VI 系统** - 参考 shadcn/ui 的 Design Token 体系  
✅ **直接生成代码** - 输出可执行的 JSX 代码  
✅ **实时预览** - 左右分屏，边生成边预览  
✅ **专业设计** - 统一的视觉语言和组件风格

## 🐛 遇到问题？

### 生成失败
1. 检查网络连接
2. 查看浏览器控制台错误
3. 尝试简化需求描述
4. 点击"重试"按钮

### 渲染错误
1. 检查浏览器控制台
2. 确保使用现代浏览器（Chrome/Edge/Firefox 最新版）
3. 尝试重新生成

### 其他问题
1. 切换到 V1 版本试试
2. 查看详细文档
3. 提交 Issue 反馈

## 🎉 开始使用

现在就打开应用，输入你的第一个看板需求吧！

```bash
npm run dev
```

访问 http://localhost:3000 开始体验 V2！
