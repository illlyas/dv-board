# 大屏设计体系 · 青蓝赛博指挥舱

> Category: 数据可视化大屏 · Big Screen  
> **暗色主题（dark UI）**：深蓝近黑底色 + 霓虹青蓝强调 + 细线发光边框，面向指挥中心 / 城市态势类 16:9 固定画布。

## 1. 视觉主题与氛围

- **场景**：固定画布 **1920×1080**，信息密度高，左—中—右或上—中—下模块化分区；中央可承载地图或主视觉，两侧为 KPI、柱状、环形、列表。
- **气质**：冷静、精密、科技感；避免温馨的圆角卡片堆叠，优先「线框 + 弱发光」界定模块。
- **模式**：全文遵循 **暗色主题**，不提供浅色主界面；大屏背景保持极低亮度，文字与数据为主视觉。

`#22D3EE` · `#38BDF8` · `#818CF8` · `#34D399` · `#FBBF24` · `#FB7185`

**大屏图表域（映射 `--dv-chart-*`）**  
- 绘图区背景极弱或与 `--color-muted` 混色透明感；网格线偏青灰低不透明；坐标轴与刻度 `#64748B` 层级；Tooltip 深色底 + 浅色字 + 青细边框。

**图表 / 表格外层壳（与生成管线对齐）**  
- `--dv-chart-panel-padding`：**0**  
- `--dv-chart-panel-border`：**none**  
- `--dv-chart-panel-radius`：**0**  
- `--dv-chart-panel-bg`：**color-mix** 弱 `--color-muted`，无「白卡片」感。

## 3. KPI 指标卡（`--kpi-*`）

不论全局 theme 命名如何，**KPI 区块固定为深色渐变卡 + 浅色数字**：  
- `--kpi-bg-from` / `--kpi-bg-to`：如 `#0F172A` → `#1E293B`，可微量混入主色色相。  
- `--kpi-text-primary` / `--kpi-text-secondary` / `--kpi-text-muted`：浅白至灰白层级；禁止浅底深字。

## 4. 字体与排版

- **标题 / 数字展示**：`"DIN Alternate", "Roboto Condensed", system-ui, sans-serif`；字重 600–700。  
- **正文 / 说明**：`system-ui, "Segoe UI", sans-serif`；字重 400–500。  
- **等宽 / 流水号**：`ui-monospace, "JetBrains Mono", monospace`。  
- **字号阶梯（px）**：12 · 14 · 16 · 18 · 22 · 28 · 36 · 48（大屏标题可用 36–48）。  
- **行高**：正文 1.45–1.55；标题 1.15–1.25。  
- **大字字母距**：标题 ≥28px 时可略收紧 `-0.02em`。

## 5. 间距、圆角与阴影

- **间距基准**：8px；阶梯 8 / 12 / 16 / 24 / 32 / 48 / 64。  
- **圆角**：模块外壳可用 **4–8px** 或 **0**（硬朗指挥风）；按钮 / 筛选控件 **6–8px**；禁止超大圆角「消费级卡片」感。  
- **阴影与发光**：以外发光为主（`color-mix` + 青色系），模拟霓虹勾边；避免厚重 Material 投影。

## 6. 布局原则（大屏）

- 画布 **1920×1080**，**overflow: hidden**，主内容区 **flex:1 + minHeight:0**。  
- 栅格：**repeat(N, minmax(0, 1fr))**；行高以 **px** 或 **minmax(0,1fr)** 规划，避免 `auto` 行撑破高度。  
- **顶栏**：固定高度约 **96px**，主标题居中，底层可叠矢量底纹；筛选区贴底右对齐。  
- 模块标题条：细下划线或左侧色条即可，避免臃肿装饰。

## 7. 组件与交互

- **筛选 / Tab**：描边 + 半透明底；选中态主色填充或主色描边。  
- **表格**：深色斑马纹可用 `--color-surface-2`；状态列可用语义色点。  
- **动效**：150–240ms，ease-out；禁止夸张弹跳；数据刷新可用轻微 opacity pulse。

## 8. 文案调性

- 简短、指令感、指标导向；避免营销口水话。  
- 模块预设标题可用「标题文本」类占位，真实项目替换为业务语义。

## 9. 反模式

- ❌ 浅色整页背景或浅色 KPI 卡（破坏大屏沉浸与 `--kpi-*` 约定）。  
- ❌ 图表外再套白底圆角重卡片（违背 `--dv-chart-panel-*`）。  
- ❌ 霓虹色大面积铺底导致眩光与 WCAG 对比失控。  
- ❌ 与「暗色主题」矛盾的 light mode 描述。
