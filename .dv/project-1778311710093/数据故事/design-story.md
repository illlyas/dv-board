# 智慧工厂生产运营数字孪生看板

## 概述
本看板以数字孪生技术为核心，整合设备、质量、能耗、安全等多维度实时数据，为一线运维人员提供“感知-诊断-决策”闭环的可视化工具。通过大屏动态展示工厂运行全景，助力管理人员快速定位异常、优化生产节拍，实现精益化与智能化管控。

## 业务目标
- **核心目标**：实时监控工厂生产全要素状态，提升设备综合效率（OEE）与良品率，降低能耗与安全风险。
- **受众**：产线组长、设备维护工程师等一线运维人员。
- **决策支持**：当设备故障、质量异常、能耗超标或安全事件发生时，支持即时干预与流程触发；基于历史对比辅助排产与库存策略调整。

## 核心指标
| 指标名称 | 类型 | 说明 |
|---------|------|------|
| 设备综合效率 (OEE) | KPI | 衡量设备利用率、性能与质量综合表现，反映产线整体效率 |
| 良品率 / 缺陷率 | KPI | 产品合格比例，缺陷率=1-良品率，直接反映质量水平 |
| 设备开机率 / 故障率 | 过程 | 开机率=设备运行时间/计划时间；故障率=故障停机时间/总时间 |
| 生产产量 / 产能达成率 | KPI | 实际产出与目标产能的比值，按小时/班次/日聚合 |
| 产线节拍 / CT时间 | 过程 | 完成一个产品所需平均时间，对比标准节拍发现瓶颈 |
| 能耗数据（水电气消耗） | 诊断 | 单位产品能耗、分区域/设备的实时能耗值 |
| 库存周转率 / 物料消耗 | 诊断 | 库存周转率=出库量/平均库存，物料消耗速率用于预警补货 |
| 安全指标（事故率 / 隐患排查数） | KPI | 事故率=事故数/工时，隐患排查数反映主动防控力度 |

## 分析维度
- **时间**：小时/班次/天/周，支持粒度切换，识别生产节奏与趋势。
- **产品类型/型号**：按不同产品追踪OEE、良品率与产能差异。
- **工厂区域/车间**：分车间（如注塑、装配、仓储）对比状态与绩效。
- **产线/工序/工位**：定位具体产线或工位的节拍、故障与质量数据。
- **设备型号/编号**：单设备运行参数、OEE及历史维护记录。
- **班组/操作人员**：按班组统计生产效率、质量与安全表现。

## 对比分析
- **基准对比**：与行业标准值（如OEE≥85%）及历史同期均值对比，标识达标与偏离。
- **趋势对比**：当前指标 vs 上一班次、昨日、上周同期，突出环比变化。

## 关键决策场景
1. **设备故障干预**：如果设备故障率超过5%或OEE低于阈值（如80%），则自动派发维修工单并通知维护工程师，同步锁定该工位产能。
2. **产能未达目标**：如果当前班次产量低于计划目标90%，则触发排产调整建议（如增加班次、切换并行产线），并显示瓶颈工序。
3. **良品率下降**：如果良品率连续下降超过2个百分点（或缺陷率上升），则启动质量排查流程，调取该时段物料批次、设备参数及操作人员信息。
4. **能耗超标**：如果某车间单位产品能耗超过基准值10%，则推送节能优化建议（如调整设备运行参数、关闭空闲设备）。
5. **安全事件**：如果发生事故或隐患排查数突破阈值（如隐患未整改超24小时），则自动报警并启动应急响应流程，通知安全专员与产线负责人。
6. **库存异常**：如果库存周转率低于设定下限或物料消耗速率异常，则触发补货/清滞预警，推荐调整采购计划。

## 预警规则
- **红色告警**：OEE＜70%、故障率＞10%、良品率＜90%、安全事故发生、单日能耗超上限20%。
- **黄色预警**：OEE 70%-80%、故障率5%-10%、良品率90%-95%、能耗超限10%-20%、隐患排查超期未整改。
- **绿色正常**：各项指标在合理范围内。

## 页面规划建议
建议分为 3 个页面（可通过大屏交互切换）：

1. **总览驾驶舱**：展示工厂3D数字孪生地图，核心KPI（OEE、良品率、产能达成率、能耗、安全）以仪表盘与轮播卡片呈现；设备状态用红/黄/绿指示灯标注；实时滚动预警列表。目的：让用户一眼掌握全局健康状况。
2. **设备与质量深度**：左侧为设备列表（按OEE排序），右侧展示选中设备的OEE分解（可用性、性能、质量）、CT时间趋势、故障频次热力图；下方为良品率按产品/班组的对比柱状图。目的：快速定位低效设备与质量薄弱环节。
3. **能耗与安全诊断**：展示水电气分时段/分区域能耗曲线，单位产品能耗对标；安全指标（事故率月趋势、隐患排查闭环率）；下方为异常事件的时间轴与关联分析。目的：辅助节能优化与安全预防。

## 数据故事线
看板从工厂整体运行状态出发，首先呈现OEE、良品率等关键KPI的实时达标情况，引导用户关注“红灯”区域。当用户点击异常设备或产线时，故事自然过渡到第二页，深入分析OEE损失原因（停机、减速、缺陷），并关联到故障工单与质量数据。最后，第三页将能耗与安全作为辅助诊断线索，揭示异常背后的长期趋势与潜在风险，最终指导一线运维人员采取精准干预措施。
