# 智慧工厂质量管控追溯看板

## 概述
本看板以智慧工厂的生产过程为核心，通过集成产量、质量、设备效率、能耗等多维数据，利用数字孪生与AI预警技术，实现从宏观生产态势到微观质量根因的实时可视化追溯。支持多角色协同监控与快速决策，体现工厂“感知-分析-响应”的先进制造能力。

## 业务目标
- **核心目标**：实现生产过程的质量追溯与持续改善，提升良品率和设备综合效率。
- **受众**：多角色混合（工厂管理层、生产主管、质量经理、设备维护工程师、班组长），每个角色可切换至对应视图。
- **决策支持**：快速定位质量异常的产线、工序、物料批次；优化生产节拍与能耗；对比目标与历史绩效，驱动改善行动。

## 核心指标
| 指标名称 | 类型 | 说明 |
|---------|------|------|
| 产量 | KPI | 当日/当班实际产出数量（件/kg） |
| 良品率 | KPI | 合格产品占总产量的百分比 |
| OEE（设备综合效率） | KPI | 设备利用率、性能效率、良品率的综合比率 |
| 生产节拍 | 过程 | 单件产品平均生产所需时间（秒/件） |
| 能耗 | 过程 | 单位产量能耗（kWh/件）或总能耗 |
| 停机时间 | 诊断 | 设备非计划停机总时长（分钟） |
| 工单完成率 | KPI | 已完工工单占计划工单的比例 |

## 分析维度
- **时间**：分钟级实时、班次、日、周、月、自定义时段，支持同比/环比钻取。
- **产线/工段/机台**：按物理层级下钻，定位异常源头。
- **产品型号**：区分不同SKU的生产表现。
- **班组**：对比不同班组的产量、良品率与效率。
- **工序**：追溯各工艺环节的质量分布与瓶颈节拍。
- **物料批次**：关联采购批次与工艺参数，定位原材料引起的质量波动。

## 对比分析
- **目标对比**：实际指标 vs 计划目标（如良品率≥99%，OEE≥85%）。
- **同比**：与去年同期的生产同期对比（如月产量同比）。
- **环比**：与上一统计周期（日/班次）的对比。
- **行业对比**：针对OEE、良品率对标行业基准值（可配置行业标准）。
- **无需对比**：支持用户在大屏上切换至仅显示绝对值模式，去除对比线。

## 关键决策场景
1. **良品率告警处置**：如果某产线良品率连续3分钟低于目标阈值（如98%），则系统自动推送预警至质量主管，并高亮该产线对应的物料批次与工序环节，触发追溯分析。
2. **异常节拍干预**：如果某工段生产节拍超出标准值20%以上，则自动发出设备负荷或产线拥堵预警，建议班组长检查机台参数或上游来料。
3. **能耗与停机关联分析**：如果停机时间异常增长且同期能耗未下降，则提示可能存在非生产空转损耗，建议设备维护工程师排查无效待机。
4. **工单延迟响应**：如果工单完成率低于80%且OEE同时下降，则提醒生产调度人员优先调整资源分配，并追溯相关班组及工序。

## 预警规则
- 良品率 < 98%（连续5分钟）：红色预警，通知质量与生产主管。
- OEE < 80%（连续1小时）：橙色预警，通知设备与生产主管。
- 停机时间 > 30分钟/小时：黄色预警，通知维护工程师。
- 单位能耗 > 目标值×1.2：黄色预警，通知能源管理员。
- 生产节拍 > 标准节拍×1.3：蓝色预警，通知班组长。

## 页面规划建议
建议分为 3 个页面（大屏轮播或按需切换）：
1. **生产全域总览**：顶部展示工厂3D数字孪生缩略图，实时滚动产量、良品率、OEE、能耗等核心KPI卡。中间为主趋势图（产量与良品率双轴拆线），底部为产线对比雷达图。用途：快速掌握全局健康度。
2. **质量追溯与根因分析**：左侧为质量异常产线/工序热力图，中间为所选异常点的物料批次与工艺参数明细表，右侧展示同类历史异常趋势。支持点击钻取至对应工段及机台视频回放。用途：定位质量问题的根本原因。
3. **设备效率与能耗诊断**：展示OEE、停机时间、生产节拍的分解树状图（按产线/机台），下方为能耗与产量散点图，可切换至班组同比对比。用途：发现低效设备与过量能耗，指导预防性维护。

## 数据故事线
从总览开始，用户首先看到工厂整体的生产绩效——产量、良品率、OEE是否达标；一旦良品率或OEE低于阈值，自动追踪至异常产线的工序与物料批次；进一步对比该批次的历史表现与当前班组参数，发现根因；最后联动设备效率与能耗数据，判断是否由设备老化或参数偏移引起，从而驱动改善行动。整个叙事遵循“全局感知→异常定位→原因追溯→行动决策”的闭环逻辑。
