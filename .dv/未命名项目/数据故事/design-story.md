# 智慧工厂生产运营看板

## 概述
本看板面向工厂管理层与车间调度人员，实时监控生产、质量、设备、能耗等核心维度，通过多维度钻取与智能预警，驱动精益生产与快速决策。适用于离散制造或流程制造企业的数字化车间。

## 业务目标
- **核心目标**：提升设备综合效率（OEE）与良品率，降低能耗与停机时间，实现生产可追溯、可预警、可调优。
- **受众**：厂长、生产主管、设备维护主管、质量工程师、调度员。
- **决策支持**：支持产能调配、设备检修排程、质量追溯、节能策略调整、班次优化。

## 核心指标
| 指标名称 | 类型 | 说明 |
|---------|------|------|
| 产量（总/日/班次） | KPI | 统计周期内生产完成数，含总产量、日产量、班次产量 |
| 良品率/合格率 | KPI | 合格品占总产出比例，反映质量水平 |
| OEE（设备综合效率） | KPI | 设备可用率×性能效率×质量率，综合衡量设备效能 |
| 设备利用率 | 过程 | 设备实际运行时间与计划运行时间的比值 |
| 能耗（电/水/气） | 过程 | 单位时间或单位产品的能源消耗量 |
| 故障率/停机时间 | 诊断 | 设备故障频次与累计停机时长 |
| 库存周转率 | 过程 | 原材料/成品周转速度，反映供应链效率 |
| 生产节拍（小时产出） | 过程 | 每小时产出数量，衡量产线速度 |

## 分析维度
- **时间**：按小时、班次、日、周下钻，观察趋势与异常时段。
- **产线/工位**：区分不同产线或工位的表现，定位瓶颈。
- **批次/订单号**：追溯特定订单或批次的质量与进度。
- **车间/区域**：按物理区域（如A车间、B车间）对比能耗与产出。
- **工序/工艺**：分析各工序合格率与节拍，发现工艺短板。
- **产品种类/型号**：区分不同产品型号的产量与良品率差异。
- **设备类型/编号**：监控单台设备的OEE、故障率及能耗。

## 对比分析
- **环比**：与上一周期（上日/上周/上月）对比，观察短期波动。
- **基准对比**：与行业标杆或历史均值对比，定位差距。
- **目标对比**：与计划值、预算值对比，评估达成情况。
- **同比**：与去年同期对比，排除季节性干扰。

## 关键决策场景
1. **设备检修触发**：如果某设备故障率连续2小时超过阈值或OEE低于70%，则自动生成检修工单，安排维护人员介入。
2. **质量追溯启动**：如果良品率低于98%且下降趋势明显，则自动触发该批次/工位/工序的质量追溯，调取历史数据与质检记录。
3. **产能调整决策**：如果当日产量低于计划值且预计当日无法追回，则启动班次调整（如增加加班班次）或排产优化（优先高优先级订单）。
4. **节能模式激活**：如果能耗（电/水/气）超过计划上限10%，则自动推送节能建议（如调低非关键设备功率）并通知能源管理员。
5. **异常告警与排查**：如果OEE、良品率、产量等核心指标同时大幅下降（如连续3个数据点偏离3σ），则自动触发全屏红色告警，要求值班人员立即排查。
6. **目标达成奖励**：如果班次产量与良品率同时达标，则自动生成奖励报告并推送至管理层审批。

## 预警规则
- OEE < 75% → 黄色预警，建议关注设备维护。
- 良品率 < 95% → 橙色预警，启动质量追溯。
- 设备故障停机 > 30分钟/班次 → 红色预警，强制派单。
- 能耗（电） > 计划值 5% → 黄色预警；>10% → 红色预警。
- 生产节拍低于目标值20% → 橙色预警，查找瓶颈工序。

## 页面规划建议
建议分为 4 个页面：
1. **全局总览**：展示当日总产量、良品率、OEE、能耗等核心KPI卡片，配合环形进度图（目标对比）和滚动环比箭头，一眼掌握工厂运行状态。
2. **产线/设备详情**：以产线或设备为单位，展示OEE、故障率、节拍趋势图，支持点击钻取到单台设备的历史曲线与故障事件列表，便于快速定位异常设备。
3. **质量与追溯看板**：聚焦良品率、缺陷分布（按工序/批次/产品型号），嵌入帕累托图、热力图，并关联批次追溯链路，便于质量工程师进行根因分析。
4. **能耗与效率诊断**：展示水电气能耗趋势、单位产品能耗对比，叠加节能模式触发历史记录，并预留“节能建议”面板，辅助能源管理员决策。

## 数据故事线
从工厂全局产能与质量切入，通过对比目标与历史数据快速识别异常；下钻至产线与设备层级，结合OEE与故障率锁定瓶颈；再深入质量与能耗维度展开根因分析，最终驱动检修、排产、节能等闭环决策。整体叙事遵循“总览→定位→诊断→行动”的逻辑，帮助管理者在30秒内掌握全局，3分钟内找到改进点。
