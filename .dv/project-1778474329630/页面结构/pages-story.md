# 智慧工厂运营看板 — 页面结构设计

## 看板概览

- **行业标签**：industrial
- **核心实体**：工厂生产（产线、设备、能耗、质量）
- **目标受众**：设备与能耗管理人员、生产管理人员、质量管理人员、运营调度人员
- **整体目标**：实时掌控工厂产、设、能、质、本关键指标，快速定位异常并驱动主动干预与决策
- **视觉基调**：operational
- **推荐主题**：dark-tech
- **布局密度**：compact
- **内容重点**：chart-first

## 数据故事线

从宏观生产进度入手（产量与计划完成率），再下探到设备效能（OEE）与能耗水平，当发现偏差时进入质量与故障细节分析（不良率、缺陷分类、停机原因），最后评估成本与运营健康度（成本、利润、库存），形成“看全局→查原因→定措施”的闭环叙事逻辑。每页均设置预警触发点，引导用户关注异常数据，推动即时决策。

## 默认分析切片

- **时间维度**：小时 / 班次 / 日 / 周 / 月
- **产线/车间维度**：按产线对比，下钻至工位
- **设备/工位维度**：设备级OEE与故障分析
- **产品品类/型号维度**：质量、产量按产品细分
- **工序/工艺维度**：不良率按工序定位
- **班组/人员维度**：产量与停机关联班组
- **地域/厂区维度**：多厂区对比（如适用）

## 核心关注点

- 产量与计划完成率是否达标（持续低于90%触发预警）
- 设备OEE是否低于目标值（85%）
- 能耗总量/单位能耗是否超计划（10%阈值）
- 不良率是否超过阈值（1%/3%）
- 设备故障率与停机时长是否异常（5%/10%阈值）
- 成本环比是否上涨（5%/10%阈值）

## 页面设计

### P1 生产与效率总览

- **故事角色**：总览
- **核心问题**：当前生产进度是否正常？设备效率与产能分布如何？
- **分析目标**：overview / trend / ranking
- **分析角度**：产量趋势、产线产能对比、设备故障排名
- **必现洞察**：产量连续2小时低于目标90%需排产调整；OEE低于85%需预防性维护；设备故障率前5名需优先处理
- **决策行动**：根据趋势与排名调整班次、安排维护、启动备产
- **核心指标**：产量、计划完成率、OEE、设备故障率、停机时长
- **数据维度**：产线、车间、设备、班组
- **筛选器**：日期范围选择器（DateRangePicker）、产线下拉选择器（Select）
- **主视觉组件**：1. 产量趋势折线图（★）

**页面叙事**：本页从左到右展示宏观生产进度与设备效能。左侧KPI卡片组实时更新产量、计划完成率与OEE，下方产量趋势图叠加计划线与预警阈值，帮助判断整体生产节奏；中栏产线OEE对比图揭示差异；右栏设备故障率与停机时长排行榜直接定位高故障设备，为维护决策提供依据。

### P2 能耗与质量监控

- **故事角色**：详情
- **核心问题**：能耗是否超限？质量缺陷分布与趋势如何？
- **分析目标**：trend / composition / comparison / target-gap
- **分析角度**：能耗时序、能耗构成、合格率趋势、缺陷分类、工序/班组差异、原材料消耗
- **必现洞察**：能耗超计划10%需启动节能；不良率超3%需停线追溯；缺陷集中在某工序须工艺改进
- **决策行动**：调整产线能耗策略、触发质量停线、优化工序工艺
- **核心指标**：能耗总量、单位能耗、产品合格率、不良率、缺陷分类、原材料消耗
- **数据维度**：时间、产线、工序、班组、产品型号
- **筛选器**：日期范围选择器（DateRangePicker）、产线下拉选择器（Select）、产品型号多选（MultiSelect）
- **主视觉组件**：3. 能耗总量与单位能耗双轴折线图（★）

**页面叙事**：本页聚焦能耗与质量两大成本与风险中心。左侧能耗时序图叠加计划线与阈值预警，构成流量核心；能耗构成饼图揭示能耗源头；中栏产品质量合格率与不良率趋势联动分析，缺陷分类堆叠图找出主要缺陷；右栏按工序/班组的质量对比柱状图定位低合格率环节，原材料消耗趋势辅助成本归因。

### P3 设备与运营诊断

- **故事角色**：诊断
- **核心问题**：设备哪些处于亚健康？成本与利润趋势如何？运营效率如何？
- **分析目标**：risk / trend / ranking / detail
- **分析角度**：设备OEE与故障率分布、报警事件、成本/利润趋势、库存周转、班组绩效、原材料消耗排行
- **必现洞察**：OEE<75%的设备需紧急维护；故障率>10%生成维修工单；成本环比升5%审查物料；班组绩效差距大需整顿
- **决策行动**：派发维修工单、调整库存策略、优化班组排班、启动成本控制
- **核心指标**：设备OEE、故障率、成本总额、利润、库存周转率、原材料消耗、班组产量
- **数据维度**：设备、时间、班组、厂区、产品
- **筛选器**：日期范围选择器（DateRangePicker）、设备类型多选（MultiSelect）、厂区/车间选择器（Select）
- **主视觉组件**：2. 设备OEE vs 故障率散点气泡图（★）— **注**：因系统组件库不支持散点气泡图，采用**双轴柱状图**（左轴OEE，右轴故障率，每设备一柱，用颜色区分状态）

**页面叙事**：本页进入深度诊断模式，综合设备状态、成本效益与运营效率。左侧设备状态双轴图（OEE与故障率对比）快速识别亚健康设备，下附报警事件滚动表突出即时风险；中栏成本与利润趋势图揭示财务健康度；右栏库存周转率趋势、班组绩效表及原材料消耗排行支撑运营优化。

## 潜在扩展需求

- **设备维保日历或工单看板**：可与报警事件联动，增加工单生命周期跟踪模块
- **能耗预测模型**：基于历史数据预测未来时段能耗，辅助节能调度
- **质量追溯详情页**：点击缺陷分类可下钻至具体批次、物料批号及操作人员
- **移动端简版看板**：关键报警与KPI推送至手机端
- **多语言切换与国际化**：满足外资或跨国工厂需求
